Add to Homescreen

What is this?

Enter a URL into the text box above and press embed. Then, on any iOS device the website can be added to your homescreen. It will open in fullscreen as if it's an app, instead of opening Safari like normal.

Some websites may show a blank white screen. Rarely, searching something like "iframe example.com refused to connect" will give an alternate URL that works (For example google.com doesn't work but google.com/search?igpu=1 does). Otherwise, this limitation can't be avoided no matter what. Sorry.
